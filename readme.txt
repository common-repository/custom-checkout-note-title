== Short Description ==
A simple plugin to change the title of the Additional Note field in WooCommerce checkout, customizable from the admin panel.

== Description ==
Custom Checkout Note Title allows you to easily change the title and placeholder text of the Additional Note field on the WooCommerce checkout page. This is useful for customizing the messaging for customers who need to add special instructions or notes to their orders. You can configure the title and placeholder directly from the WooCommerce settings panel.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/custom-checkout-note-title` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce settings > Checkout to configure.

== Frequently Asked Questions ==
= How do I change the note title? =
You can go to WooCommerce settings under the "Payments" tab go to "Custom Checkout Note Settings" section and update the custom note title.
